# run-dev — Levanta servidor Astro

Inicia el servidor de desarrollo de Más de Chile SpA en http://localhost:4321/

## Ejemplo

```bash
/run-dev
```

Lanza `npm run dev` y verifica que esté activo en http://localhost:4321/

## Notas

- Hot reload automático en cambios
- Astro v5.18.2
- Puerto 4321
- Ctrl+C para detener
